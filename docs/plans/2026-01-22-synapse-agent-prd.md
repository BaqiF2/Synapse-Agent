## 快速导航

### 核心理念
- **已知定律**：工具是 Agent 的手脚、文件系统是记忆载体、技能的本质
- **待验证假设**：统一抽象层、功能即工具、智能的涌现、通用架构公式、沙盒的定位

### 项目目标
- 理念验证：通过可运行的原型证明核心假设的可行性
- 自我成长：实现 Agent 自动扩展技能库、强化已有技能
- 通用框架：为未来多种交互方式奠定基础

### 分阶段实施
- **阶段 1（8-12周）**：核心验证 - 三层 Bash、工具转换、基础 Agent Loop
- **阶段 2（10-14周）**：自我成长 - 技能搜索、技能强化、外源集成
- **阶段 3（12-16周）**：完整能力 - RLHF、性能优化、沙盒隔离（可选）
